🔒 Политика конфиденциальности

✕

Последнее обновление: май 2025

1. Общие положения

Настоящая политика описывает как сервис YuPay (@YuPay_Ex) собирает, использует и хранит персональные данные пользователей Telegram Mini App. Используя сервис, вы соглашаетесь с условиями данной политики.

2. Какие данные мы собираем

• Имя пользователя Telegram (username)
• Отображаемое имя
• История заявок на обмен (валютная пара, суммы, статус, дата)
• Дата регистрации в сервисе

Мы не собираем платёжные данные, номера карт и другую финансовую информацию.

3. Зачем мы это собираем

• Для идентификации пользователя в системе
• Для обработки заявок на обмен валюты
• Для отображения истории ваших операций
• Для связи менеджера с вами по заявке

4. Где хранятся данные

Данные хранятся на серверах Supabase (Европейский союз, West EU Paris). Supabase соответствует стандартам GDPR.

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши данные третьим лицам. Данные доступны только менеджерам YuPay для обработки заявок.

6. Удаление данных

Вы можете запросить удаление всех данных написав менеджеру: @YuPayManager. Данные будут удалены в течение 3 рабочих дней.

7. Контакты

По всем вопросам: @YuPayManager

📄 Пользовательское соглашение

✕

Последнее обновление: май 2026

1. Общие положения

Настоящее соглашение регулирует отношения между сервисом YuPay (@YuPay_Ex) и пользователем при использовании Telegram Mini App. Используя сервис, вы принимаете условия соглашения.

2. Описание сервиса

YuPay — сервис обмена валюты (рубли, юани, доллары). Сервис не является банком или платёжной системой.

3. Условия использования

• Предоставлять достоверные данные при регистрации
• Не использовать сервис в мошеннических целях
• Не создавать несколько аккаунтов
• Не нарушать работу сервиса

4. Заявки на обмен

• Заявка является намерением, а не гарантией обмена
• Курс фиксируется при подтверждении менеджером
• Сервис вправе отказать в обработке заявки

5. Ответственность

Сервис не несёт ответственности за убытки от колебаний курса валют и технические сбои на стороне Telegram. Пользователь самостоятельно соблюдает валютное законодательство своей страны.

6. Блокировка

Сервис вправе заблокировать пользователя при нарушении условий соглашения.

7. Контакты

По всем вопросам: @YuPayManager
